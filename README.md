# Çengel Bulmaca

**Türk edebiyatı ve kültürü temalı, tüm platformlarda çalışan interaktif çengel bulmaca oyunu.**

Çengel Bulmaca, oyunculara eğlenceli ve eğitici bir deneyim sunan modern bir bulmaca oyunudur. Flutterle geliştirilen uygulama, Windows, Android, iOS (iOS şuanda kısıtlamalardan ötürü indirilememekte. Web üzerinden erişilebilir.) ve web platformlarında sorunsuz çalışırken, aynı zamanda internet bağlantısı gerektirmeden offline olarak oynanabilir.

## Temel Özellikler

• **Cross-platform desteği**: Masaüstü, mobil ve web ortamlarında oynanabilir
• **Offline oynanış**: İnternet bağlantısı olmadan tam işlevsel oyun deneyimi
• **Zengin içerik sistemi**: Farklı konularda, ayarlanabilir zorluk seviyelerine sahip yüzlerce bulmaca
• **Çengel bulmaca mekanizması**: Oyuncular eksik harfleri tahmin ederek cevabı tamamlarlar
• **İpucu sistemi**: Oyuncuların ilerleme kaydedebilmeleri için ek harf ipuçları
• **Puan sistemi**: Zorluk seviyesine göre dinamik puan hesaplama

## Teknik Yapı

- Uygulama **Flutter** ile geliştirilmiş, modern state management (Provider) ve veri saklama çözümleri kullanmaktadır. Çok platformlu destek için native entegrasyonlar bulunmakta, mobil ortamda sorunsuz performans sağlanmaktadır.

---

**Çengel Bulmaca**, eğlence ve öğrenmeyi bir araya getiren, Türk edebiyatı severleri için tasarlanmış, tamamen offline çalışan bir oyundur. Yalnızca Web sürümü internet gerektirebilir.
